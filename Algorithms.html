1. Linear Search


For i = 0 to n - 1
    If array[i] == target
        Return i
Return -1

2. Binary Search


Set low = 0, high = n - 1
While low ≤ high
    mid = (low + high) / 2
    If array[mid] == target
        Return mid
    Else if array[mid] < target
        low = mid + 1
    Else
        high = mid - 1
Return -1

3. Bubble Sort


For i = 0 to n - 1
    For j = 0 to n - i - 2
        If array[j] > array[j + 1]
            Swap array[j] and array[j + 1]

4. Insertion Sort


For i = 1 to n - 1
    key = array[i]
    j = i - 1
    While j ≥ 0 and array[j] > key
        array[j + 1] = array[j]
        j = j - 1
    array[j + 1] = key

5. Selection Sort


For i = 0 to n - 1
    minIndex = i
    For j = i + 1 to n - 1
        If array[j] < array[minIndex]
            minIndex = j
    Swap array[i] and array[minIndex]

6. Merge Sort


If length(array) ≤ 1
    Return array
Split array into left and right halves
Left = MergeSort(left)
Right = MergeSort(right)
Merge Left and Right

7. Quick Sort


If low < high
    pivot = Partition(array, low, high)
    QuickSort(array, low, pivot - 1)
    QuickSort(array, pivot + 1, high)

8. Heap Sort


Build a max heap
For i = n - 1 down to 1
    Swap array[0] and array[i]
    Heapify heap of size i

9. Dijkstra’s Algorithm


Initialize all distances to infinity
Set source distance to 0
While unvisited vertices remain
    Select vertex with minimum distance
    Update distances of its neighbors

10. Fibonacci Sequence


Set a = 0, b = 1
Print a, b
For i = 3 to n
    c = a + b
    Print c
    a = b
    b = c
