1. Binary Search


1. Set low = 0, high = n - 1
2. While low ≤ high
3.     mid = (low + high) / 2
4.     If array[mid] == target, return mid
5.     Else if array[mid] < target, low = mid + 1
6.     Else high = mid - 1
7. Return -1

2. Bubble Sort


1. For i = 0 to n - 1
2.     For j = 0 to n - i - 2
3.         If array[j] > array[j + 1]
4.             Swap array[j] and array[j + 1]

3. Selection Sort


1. For i = 0 to n - 1
2.     minIndex = i
3.     For j = i + 1 to n - 1
4.         If array[j] < array[minIndex]
5.             minIndex = j
6.     Swap array[i] and array[minIndex]

4. Dijkstra’s Algorithm


1. Set distance of all vertices to infinity
2. Set distance of source vertex to 0
3. While unvisited vertices exist
4.     Select vertex u with minimum distance
5.     Mark u as visited
6.     For each neighbor v of u
7.         If distance[u] + weight(u,v) < distance[v]
8.             Update distance[v]

5. Depth First Search (DFS)


1. Mark current node as visited
2. For each adjacent node not visited
3.     Call DFS recursively

6. Breadth First Search (BFS)


1. Enqueue starting node and mark as visited
2. While queue is not empty
3.     Dequeue a node and visit it
4.     Enqueue all unvisited adjacent nodes

7. Floyd–Warshall Algorithm


1. Initialize distance matrix
2. For k = 0 to n - 1
3.     For i = 0 to n - 1
4.         For j = 0 to n - 1
5.             If dist[i][k] + dist[k][j] < dist[i][j]
6.                 dist[i][j] = dist[i][k] + dist[k][j]

8. Heap Sort


1. Build a max heap from the array
2. For i = n - 1 down to 1
3.     Swap array[0] and array[i]
4.     Heapify reduced heap

9. Insertion Sort


1. For i = 1 to n - 1
2.     key = array[i]
3.     j = i - 1
4.     While j ≥ 0 and array[j] > key
5.         array[j + 1] = array[j]
6.         j = j - 1
7.     array[j + 1] = key

10. Fibonacci Sequence


1. Set a = 0, b = 1
2. Print a, b
3. For i = 3 to n
4.     c = a + b
5.     Print c
6.     a = b, b = c
